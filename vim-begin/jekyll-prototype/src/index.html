Welcome to the Vim Beginners’ Site. This site aims to link to and host the best online and offline resources for learning the The Vim Text Editor, an open-source, cross-platform, powerful and configurable, yet not too difficult to become productive with, text editor.

Due to the proliferation of various resources on the Internet, this site is, while open content/free content (under the Creative Commons Attribution Licence), is centrally managed and controlled, in order to make sure it refers to resources that are of good quality. It aims to be for Vim what The Perl Beginner Site’ is for Perl.
