This site is a mostly static HTML site generated by the Jekyll offline content management system based on Ruby, by SASS/compass and by other technologies.

Version Control Repository

The most up-to-date source code for this site is now hosted in a Mercurial (hg) repository on Bitbucket.org. In order to learn how to use Mercurial, see either one of:

  1. "Mercurial: The Definitive Guide" (The hgbook)

  2. Hg Init: a Mercurial tutorial.

  3. "A Tutorial on Using Mercurial" - on the Mercurial wiki.

After that, you can clone the repository on Bitbucket, and send us pull requests.
