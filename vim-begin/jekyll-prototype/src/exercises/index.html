Vim Golf

In code golf one has to solve a given problem using the shortest amount of code (usually measured in bytes). Golf can teach you many interesting and useful tricks, but can be incredibly addictive (you have been warned).

Here are some site for playing Vim Golf:

  • VimGolf - “Real Vim ninjas count every keystroeke — do you?”

  • Anarcy Golf - a multi-language golf, which also supports vim.
